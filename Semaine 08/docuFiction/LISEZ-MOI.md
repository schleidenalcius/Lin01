# Mission : Docu Fiction
## Livrable
+ une page index.html
+ trois pages enfants formatées à l'identique (cthulhu, ghibli, samurai)
+ un fichier style.css
+ un logo pour le site
## Objectif
Vous devez créer le mini-site Docu Fiction. Le site comportera une page d'accueil et trois pages enfants. On doit pouvoir aisément naviguer d'une page à n'importe quelle autre sur le site, e.g. passer de Cthulhu à Samurai à la page d'accueil avec le moins de clics posibles. La page d'accueil n'existant pas, vous devrez donc la créer selon vos goûts tout en vous assurant qu'elle invite à visiter les pages enfants. Comme il s'agit d'un site web, le style doit être constant sur toutes les pages. Vous devez également créer le fichier style.css.
Les pages enfants ne sont pas uniformes, donc vous devrez les modifier pour les uniformiser, e.g. même en-tête, même menu de navigation, etc. Les sections n'ont pas à être identiques mais les fichiers doivent avoir la même structure générale (header, main, section, etc.).
